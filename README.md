# Teacher Nel's Website

Static GitHub Pages version of the original Google Site.

## Update Content

- Edit `data.js` to change page text, links, stats, and resource cards.
- Replace images in `assets/` and update the paths in `data.js` or `styles.css`.
- Edit `styles.css` for colors, spacing, and layout.

## Publish on GitHub Pages

1. Push this folder to a GitHub repository.
2. In the repository, open Settings → Pages.
3. Choose `Deploy from a branch`.
4. Select the `main` branch and root folder.
5. Save, then wait for GitHub to publish the site.

The entry file is `index.html`.
